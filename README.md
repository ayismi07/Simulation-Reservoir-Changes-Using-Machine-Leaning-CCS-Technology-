# Simulation-Reservoir-Changes-Using-Machine-Leaning-CCS-Technology-
A data-driven project simulating reservoir dynamic changes before and after CO₂ injection. It combines a Bayesian Markov Chain Monte Carlo (MCMC) algorithm and Convolutional Neural Network (CNN) to predict physical property shifts (density, porosity, vp, water saturation) efficiently with lower cost.
